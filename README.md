- WALAB_Project2는 CRUD(Create, Read, Update, Delete) 기능과 파일 I/O를 통해 데이터를 관리하는 Java 프로젝트입니다.

- 인프런 홈페이지의 강의, 유저 데이터 관리를 콘솔 프로그램으로 제작하였습니다. https://www.inflearn.com/courses?types=ONLINE
